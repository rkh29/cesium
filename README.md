# 卫星异常检测系统

## 项目概述

卫星异常检测系统是一款基于Vue 3和ECharts开发的实时监控平台，通过可视化方式展示卫星轨道和运行状态，实现异常自动检测与告警推送，解决了传统卫星监控系统数据复杂、异常发现不及时的问题。核心创意在于将复杂的卫星数据转化为直观的可视化图表，结合智能告警机制，大幅提升卫星运维效率。

## 目标人群

- 卫星地面控制中心运维人员
- 航天工程技术人员
- 卫星数据分析师
- 航天科研机构研究人员
- 卫星运营管理团队

## 解决的问题

1. **实时监控难题**：提供24小时不间断的卫星运行状态监测，通过可视化地图直观展示卫星轨道和位置
2. **异常发现滞后**：智能分析卫星数据，实时推送不同紧急程度的告警信息，大幅缩短异常响应时间
3. **数据理解复杂**：将海量卫星数据转化为易懂的图表和指标，降低数据分析门槛
4. **运维效率低下**：集成卫星管理、规则配置、数据查询等功能，实现一站式运维管理
5. **异常溯源困难**：提供完整的异常事件链路分析，帮助快速定位问题根源

## 技术亮点

- 基于Vue 3 Composition API构建，代码结构清晰，易于维护
- 采用ECharts实现高质量卫星轨道可视化
- 响应式设计，适配各种屏幕尺寸
- 支持自定义检测规则和告警阈值
- 完善的异常分析和趋势预测功能
